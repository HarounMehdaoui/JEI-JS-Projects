# JEI-JS-Projects

## To-Do App Updates

- Due dates: Every task now requires a due date.
- Add tasks with a title and a date using the new date picker.
- Tasks persist in `localStorage` including their due dates.
- Existing saved tasks are migrated automatically and assigned today's date; you can edit them to adjust.

### Usage

1. Type a task name.
2. Pick a due date.
3. Click the plus button or press Enter to add.
